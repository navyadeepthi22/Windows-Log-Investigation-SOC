# Windows Log Investigation for SOC Analysts

This project focuses on investigating Windows authentication logs to detect suspicious login and brute-force activity.

## Objective
To analyze Windows Security Event logs and identify abnormal authentication behavior relevant to SOC investigations.

## Skills Demonstrated
- Windows Event Log analysis
- Authentication log investigation
- Event correlation
- Brute force detection logic
- MITRE ATT&CK mapping
- SOC escalation decisions

## Key Event IDs
- 4624 – Successful login
- 4625 – Failed login
- 4634 – Logoff
- 4672 – Special privileges assigned
- 4720 – User account creation

## Outcome
Suspicious authentication activity was identified and escalated as a true positive based on event correlation and access patterns.
